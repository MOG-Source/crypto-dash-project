# Crypto Dash

A modern cryptocurrency dashboard built with React and Vite.

The application displays real-time cryptocurrency data, interactive charts, market statistics, and detailed coin information through a clean and responsive interface.

## Features

- Real-time cryptocurrency market data
- Interactive charts and statistics
- Dynamic routing with React Router
- Loading spinners and async data handling
- Responsive UI design
- State management using React hooks
- API data fetching with useEffect
- Reusable React components
- Search and filtering functionality

## Technologies Used

- React
- Vite
- React Router
- JavaScript (ES6+)
- CSS
- Chart libraries
- REST APIs

## React Concepts Practiced

- useState
- useEffect
- Component-based architecture
- Props and state management
- Conditional rendering
- Dynamic routing
- API integration
- Loading and error handling

## Installation

Clone the repository:
git clone https://github.com/MOG-Source/crypto-dash-project.git

Navigate into the project folder:
cd crypto-dash-project

Install dependencies:
npm install

Run the development server:
npm run dev

## Project Purpose

This project was created to practice building modern React applications using APIs, routing, charts, and dynamic UI rendering.
